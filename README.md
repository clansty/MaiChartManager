## 目前已经实现的功能

- [x] 浏览本地谱面列表
- [x] 修改谱面基础信息
- [x] 流派 / 版本分类查看和修改
- [x] 修改乐曲音频，在音频中增加延迟或裁剪
- [x] 检查数据中潜在的问题
- [x] 导入 Simai 谱面
- [x] 软件内预览谱面
- [x] 服务器模式运行
